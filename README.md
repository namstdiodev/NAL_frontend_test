## 3. Install

### System Requirements:

- Node.js 12.0 or later
- MacOS, Windows (including WSL), and Linux are supported
- Yarn ^1.12.0

### Install:

1. Install node_modules:

```
yarn
```

2. Running dev:

```
yarn dev
```
